RawTherapee
Open Source εφαρμογή επεξεργασίας εικόνων η οποία σας παρέχει μια μεγάλη ποικιλία από τρόπους-λύσεις ώστε να ενισχύσετε τις ψηφιακές σας φωτογραφίες αλλά και άλλες εικόνες. Όπως υποδηλώνει και το όνομα, το πρόγραμμα προορίζεται για χρήση αρχείων από ψηφιακές φωτογραφικές μηχανές, αλλά μπορεί να λειτουργήσει επίσης και με άλλους τύπους αρχείων εικόνας όπως είναι τα JPEG, TIFF και PNG. 


To RawTherapee είναι ένα σύγχρονο πρόγραμμα, και ως τέτοιοo θα πρέπει να λειτουργεί με σύγχρονο υλικό. Αυτό ισχύει ιδιαίτερα εάν χρησιμοποιείτε αρχεία υψηλής ανάλυσης. Διαθέτει κάποια φίλτρα που απαιτούν πολλή μνήμη, πράγμα που σημαίνει ότι θα πρέπει να γνωρίζετε μερικά πράγματα για να αποφύγετε προβλήματα μνήμης.


WinSCP 


WinSCP is a reliable Open Source SFTP and FTP client that allows the safe copying of files between a local and a remote computer.
It is integrated with Windows and fully supports drag n' drop, URL, shortcut icons, U3, SFTP, plain old FTP protocol, SCP protocols over SSH-1/2, and Legacy SCP protocol. You can access batch file scripting and command-line interface as an alternative to the graphical user interface. It also supports multiple languages, including Chinese, French, German, and Czech.
